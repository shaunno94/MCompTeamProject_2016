Detailed Description

Random number generator for 64bit numbers.

Classes RNG32 and RNG64 can be instantiated for thread safety, but a static instance is provided for easy use in a single thread application (RNG32::Rand or RNG64::Rand).

This class uses Mersenne Twister number generator engine that produces deterministic results. The same seed can be used for several instances to produce identical number sequences and RNG64::discard can be used to synch up instances that have generated different amount of numbers.

Usage of the provided static instance, where min and max defines a range for generating numbers.

RNG64::Rand(min, max);

Usage of instantiated class, where min and max defines a range for generating numbers.

RNG64 rand(RNG64::RandSeed());
rand(min, max);
