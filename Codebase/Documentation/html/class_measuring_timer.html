Prints time periods starting at given nested level.

Parameters
osOutput stream.
startLevelThe lowest index level of period nesting to output measurements for.
maxDepthThe max depth of child periods to output.
timeResolutionTime period resolution, 1 = seconds and 1000000 = microseconds.

If parameter maxDepth is 0, only the matched start level will be printed.

record* MeasuringTimer::m_logs
private

Heaped array of MeasuringTimer::record, representing all possible time measurements.

The size of this heaped array is twice the size given to the MeasuringTimer::Timer(unsigned int) constructor, for logging start and end timestamps and producing a given number of measurable periods.
